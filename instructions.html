Instructions

1. How To reveal/edit the side contact form?

1. Go to the Navigator Panel.

2. Click on the Side Menu element.

3. Go to the Style Panel and change its setting from Display None (eye strikethrough) to Display Block (rectangle).

4. Done! Don't forget to set again the settings to Display None when you finish.

2.How To reveal/edit the fixed navbar?

As you probably know, there are 2 Navbars in order to have the effect that shows the fixed navbar when you scroll down. One is "fixed" and the other is not.

To edit the fixed one, follow these easy steps.

1. On home page, scroll down a little bit.

2. Go to the Navigator Panel and double click on "Fixed Navbar".

3. Go to the Style Panel and change its setting from Display None (eye strikethrough) to Display Block (rectangle).

4. Done! Don't forget to set again the settings to Display None when you finish.
